# PowerFit Gym Website

A professional, modern, and responsive website for PowerFit Gym. Built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Smooth Scrolling**: Enhanced navigation with smooth scrolling effects
- **Interactive Elements**: Mobile hamburger menu, animated statistics, and form validation
- **Professional Theme**: Dark navigation with red accent colors, perfect for a gym brand

## Sections

1. **Hero Section**: Eye-catching landing area with call-to-action buttons
2. **About Section**: Information about the gym with impressive statistics
3. **Services Section**: Showcase of gym services and amenities
4. **Trainers Section**: Introduction to expert trainers
5. **Contact Section**: Contact information and inquiry form
6. **Footer**: Additional links and social media information

## Getting Started

1. Open `index.html` in your web browser
2. No build process or dependencies required - it's a pure HTML/CSS/JS website

## Customization

### Colors
The color scheme can be easily customized by modifying the CSS variables in `styles.css`:
- `--primary-color`: Main accent color (default: #ff4444 - red)
- `--secondary-color`: Dark background color (default: #1a1a1a)
- `--text-color`: Main text color (default: #333)

### Content
All content can be edited directly in `index.html`. Simply modify the text, add/remove sections, or update contact information as needed.

### Images
The hero section uses a background image from Unsplash. You can replace it by:
1. Changing the URL in the `hero` section's `background` property in `styles.css`
2. Or adding your own image to an `images` folder and updating the path
